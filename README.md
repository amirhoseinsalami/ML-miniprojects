# ML Mini-Projects

A collection of machine learning and data analysis projects demonstrating practical applications of statistical modeling, visualization, and exploratory data analysis.

## Projects

### 1. House Price Prediction
Predictive modeling for California housing prices using regression techniques.

**Tech Stack:** Python, Scikit-learn, Pandas, NumPy, Matplotlib

**Models:**
- Linear Regression
- Random Forest Regressor

**Key Features:**
- Data preprocessing and feature engineering
- Model comparison using $R^2$ and RMSE metrics
- Feature importance analysis
- Visualization of predictions vs actual values

[View Project →](./house_price_prediction)

---

### 2. Interactive COVID-19 Dashboard
Comprehensive data analysis and visualization dashboard for COVID-19 statistics.

**Tech Stack:** Python, Plotly, Seaborn, Matplotlib, Pandas

**Analysis Includes:**
- Time series analysis with 7-day rolling averages
- Mortality rate calculations and comparisons
- Interactive visualizations (scatter, heatmap, boxplot, violin plot)
- Statistical analysis using regression plots and pair plots
- Geographic distribution analysis

[View Project →](./interactive_dashboard_for_covid19)

## Setup
```bash
# Clone the repository
git clone https://github.com/amirhoseinsalami/ML-miniprojects.git
cd ML-miniprojects
```

#### Navigate to a specific project
```bash
cd house_price_prediction
```
#### or
```bash
cd interactive_dashboard_for_covid19
```

Each project folder contains its own README with detailed instructions.
