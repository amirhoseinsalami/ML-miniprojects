# Interactive Dashboard for COVID-19

A data visualization dashboard for analyzing COVID-19 trends using interactive and static charts.

## Libraries Used
- `NumPy` — numerical computations
- `Pandas` — data loading and manipulation
- `Seaborn` — statistical visualizations
- `Matplotlib` — static plots
- `Plotly` — interactive charts

## Features
- Exploratory data analysis on COVID-19 dataset
- Static visualizations (heatmaps, bar charts, line plots)
- Interactive charts with Plotly (hover, zoom, filter)

## Dataset
COVID-19 public dataset containing cases, deaths, and recoveries by country/date.

## How to Run
1. Clone the repository:
```bash
   git clone https://github.com/amirhoseinsalami/ML-miniprojects.git   
```

2. Install dependencies:
```bash
   pip install numpy pandas seaborn matplotlib plotly
```   

3. Run the notebook or script:
```bash
   jupyter notebook interactive_dashboard_for_covid19.ipynb
```
   
## Results
- Visualized global COVID-19 spread over time
- Compared case trends across countries
- Identified peak periods using time-series analysis
